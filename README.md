# BDI v2.5 — Builder Density Instrument

Production-grade cognitive profiling system. AI-powered dialogue engine for measuring builder density across 3 dimensions.

## Live Demo
https://sukaczev.top/gf-test.html

## What it does
- Multi-phase state machine: identity → fluid IQ test → core probe → report
- 3D scoring: concept compression, cognitive honesty, cross-domain resonance
- Real-time leaderboard with statistical aggregation
- Session persistence and full conversation history

## Stack
Node.js, Express, PostgreSQL, REST API, DashScope AI, Nginx, PM2
